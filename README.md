# Claude_SandPit_Reference_Check

A Claude Code harness for **checking the referencing of a draft manuscript** before submission —
citations, the reference list, and quotations. It reports and proposes; it does nothing else (no
writing, no analysis).

## Documentation

- **[MANUAL.md](MANUAL.md) — the full operating manual.** Start here: the eight checks, a
  workflow diagram, running a job step by step, the **EndNote** round-trip for pulling PDFs
  through institutional access, outputs, and conventions.
- [.claude/CLAUDE.md](.claude/CLAUDE.md) — project rules read at session start (the eight tasks,
  the skills, file conventions, memory, the quotation rules, and the safety hook).
